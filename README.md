# Electric Sun Tanning

Modern tanning salon website featuring online booking, service management, Instagram-powered news updates, and an admin dashboard for business analytics.

## Features

### Customer Experience
- Browse tanning services
- Online booking flow
- About page
- Contact page
- News feed powered by Instagram content
- Responsive design for desktop and mobile

### Admin Dashboard
- Revenue tracking
- Client management
- Booking management
- Service management
- Revenue analytics and reporting
- Member activity tracking

## Tech Stack

- React
- TypeScript
- Vite
- Node.js
- Tailwind CSS
- Framer Motion
- YAML Configuration

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
