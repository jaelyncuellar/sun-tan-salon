# ELECTRICSUNTANNING — Brand & Design Guidelines

## Stance
Warm luxury boutique. Inspired by Aesop, Le Labo, and sun-drenched Mediterranean hospitality. Unhurried, sensory, deeply considered.

## Color Palette
| Token | Value | Use |
|---|---|---|
| Background | `#FBF3E8` | Warm cream page ground |
| Foreground | `#1C0D00` | Deep espresso text |
| Primary | `#B86820` | Amber — primary actions |
| Accent | `#D4952A` | Gold — highlights |
| Muted fg | `#C8A87A` | Warm tan — secondary text |
| Dark canvas | `#1C0D00` | Hero and feature sections |

## Typography
- **Display**: Playfair Display italic — all major headlines
- **Body / UI**: DM Sans 300–500 — labels, body, nav
- **Data**: DM Mono — admin IDs, timestamps

## Pages
1. **Hero** — full-bleed image, italic serif headline, dark warm ground
2. **Services** — 2-col grid with photo + features + price + CTA
3. **Booking** — 3-step wizard: service → date/time → details → confirmation
4. **About** — story, values grid, team
5. **News** — Instagram-style photo grid with modal
6. **Contact** — form + info + hours
7. **Admin** — dark sidebar dashboard: KPIs, revenue chart, pie chart, bookings table

## Voice
Short, sensory, slightly poetic. Second person. Sentences under 20 words. No clinical language.
